# Unihack - Campfire

## Setup

### Prerequisites
- Yarn ([Install here](https://classic.yarnpkg.com/en/docs/install))
- Node ([Download here](https://nodejs.org/en/))

### Install dependencies

Run (MacOS, Linux):
```
git clone https://github.com/PIG208/Campfire.git
cd Campfire
tools/setup
```

## Deployment

Run (MacOS, Linux):

git clone https://github.com/PIG208/Campfire.git
